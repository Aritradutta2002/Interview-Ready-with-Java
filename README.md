## 🗂️ CORE_JAVA Contents

### 01. Core Java OOP
- Encapsulation, Inheritance, Polymorphism, Abstraction
- Interface vs Abstract Class
- Java Keywords (static, final, super, this, etc.)

### 02. Java Memory Model
- Heap vs Stack Memory
- JVM Architecture
- Garbage Collection (GC algorithms, generations)
- Memory leaks and optimization

### 03. Exception Handling
- Checked vs Unchecked Exceptions
- Try-catch-finally
- Custom exceptions
- Best practices

### 04. Collections Framework
- List: ArrayList, LinkedList, Vector
- Map: HashMap, LinkedHashMap, TreeMap, Hashtable
- Set: HashSet, LinkedHashSet, TreeSet
- Queue: PriorityQueue, Deque

### 05. Generics
- Generic classes and methods
- Bounded types
- Wildcards (? extends, ? super)

### 06. Java 8 Features
- Lambda Expressions
- Functional Interfaces
- Stream API
- Optional class
- CompletableFuture

### 07. Java 11-17-21 Features
- Local Variable Type Inference (var)
- Text Blocks
- Switch Expressions
- Records
- Pattern Matching
- Virtual Threads (Java 21)

### 08. Multithreading & Concurrency
- Thread creation and lifecycle
- Executor Framework
- Synchronization (synchronized, locks)
- Atomic variables
- Concurrent collections

### 09. Design Patterns
- Creational: Singleton, Factory, Builder
- Structural: Adapter, Decorator, Proxy
- Behavioral: Observer, Strategy, Command

### 10. SOLID Principles
- Single Responsibility
- Open/Closed
- Liskov Substitution
- Interface Segregation
- Dependency Inversion
