## Interview Tips by Experience Level

### Junior (0-2 years)
- Focus: Core Java, OOP, Collections, Basic Multithreading
- Depth: Understand "what" and "how"

### Mid-Level (2-5 years)
- Focus: Java 8+ Features, JVM Internals, Concurrency, Design Patterns
- Depth: Understand "why" and trade-offs

### Senior (5+ years)
- Focus: Advanced Concurrency, Performance Tuning, Architecture, SOLID
- Depth: Understand internals, edge cases, and real-world scenarios

## Most Asked Topics in FAANG

1. **HashMap Internal Working** - Almost every interview
2. **ConcurrentHashMap vs HashMap** - Thread safety questions
3. **Java 8 Stream API** - Coding rounds
4. **Multithreading** - Concurrency scenarios
5. **Design Patterns** - Singleton, Factory, Builder
6. **Garbage Collection** - JVM questions
7. **CompletableFuture** - Async programming
